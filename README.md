# Instatalking

The simple chat based on ActionCable websockets

### Requirements

`ruby >= 2.5.0`

`rails ~> 5.2.0`

### Installation & using

In `terminal` or `cmd`

- `$ git clone repository_link`

- `$ gem install bundler` - if needed

- `$ bundle install`

- `$ rails s` for server starting

Then open `http://localhost:3000` at your web-browser
